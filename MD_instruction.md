# Инструкция по Markdown

## Выделение текста

Для того, чтобы выделить текст курсивом, необходимо обрамить текст звёздочками (*) или нижним подчёркиванием (_) - \**Текст выделен курсивом.*\* | \__Текст выделен курсивом._\_
